# Fasd Puppet Module for Boxen

This is a boxen module which installs [fasd](https://github.com/clvv/fasd) through brew

## Usage

```puppet
include fasd
```

## Required Puppet Modules

* `boxen`
* `homebrew`

## Development

Write code. Run one of the following tasks to test it:
* `rake syntax`
* `rake style`
* `rake spec`
* `rake build`
